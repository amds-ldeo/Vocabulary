[]{#channelmaster}

##  ChannelMaster
- Teledyne Rd Instruments model name for an ADCP designed to use
horizontal profiling for flow monitoring in inland waterways.
- Concept URI: http://mmisw.org/ont/trdi/models/ChannelMaster



[]{#horizontaladcp}

##  HorizontalADCP
- Instrument designed to measure velocity profiles in a horizontal
plane.
- Concept URI: http://mmisw.org/ont/trdi/models/HorizontalADCP



[]{#longranger}

##  LongRanger
- Model name for Teledyne RD Instruments Workhorse ADCPs of frequency
75 kHz.
- Concept URI: http://mmisw.org/ont/trdi/models/LongRanger



[]{#mariner}

##  Mariner
- Model name for a Teledyne RD Instruments Workhorse Monitor ADCP
configured for underway current measurement in shallow water (as
opposed to the deep water Ocean Surveyor systems).
- Concept URI: http://mmisw.org/ont/trdi/models/Mariner



[]{#monitor}

##  Monitor
- Teledyne RD Instruments model name for a direct reading Workhorse
ADCP.
- Concept URI: http://mmisw.org/ont/trdi/models/Monitor



[]{#navigator}

##  Navigator
- Teledyne RD Instruments model name for the Teledyne RDI Doppler
Velocity Log.
- Concept URI: http://mmisw.org/ont/trdi/models/Navigator



[]{#oceanobserver}

##  OceanObserver
- Teledyne RD Instruments model name for a low frequency Phased Array
ADCP for cabled deployment, usually from an oil rig.
- Concept URI: http://mmisw.org/ont/trdi/models/OceanObserver



[]{#oceansurveyor}

##  OceanSurveyor
- Teledyne RD Instruments model name for a low frequency Phased Array
ADCP for vessel mounted operations.
- Concept URI: http://mmisw.org/ont/trdi/models/OceanSurveyor



[]{#riograndeadcp}

##  RioGrandeADCP
- Teledyne RD Instruments model name for an ADCP optimally configured
for measurement of river discharge. A distinguishing feature of Rio
Grande ADCPs is that they are designed to operate from a 12 VDC power
supply (all other ADCPs operate from 20 - 60 VDC).
- Concept URI: http://mmisw.org/ont/trdi/models/RioGrandeADCP



[]{#sentinel}

##  Sentinel
- Teledyne RD Instruments model name for a self-contained Workhorse
ADCP.
- Concept URI: http://mmisw.org/ont/trdi/models/Sentinel



[]{#streampro}

##  StreamPro
- Teledyne RD Instruments model name for a miniaturized ADCP packaged
into a streamlined float designed to be towed by hand across very
shallow streams for quick and efficient discharge measurements.
- Concept URI: http://mmisw.org/ont/trdi/models/StreamPro



[]{#vesselmount}

##  VesselMount
- An ADCP mounted to the hull of a vessel, typically in a sea chest,
and having inputs from the vessel's navigation equipment.
- Concept URI: http://mmisw.org/ont/trdi/models/VesselMount



[]{#workhorseadcp}

##  WorkhorseADCP
- The generic model name for all of the non-phased array broadband
ADCPs currently produced by Teledyne RD Instruments
- Concept URI: http://mmisw.org/ont/trdi/models/WorkhorseADCP



[]{#zedhead}

##  ZedHead
- Teledyne RD Instruments trade name for a transducer designed to
minimize ringing.
- Concept URI: http://mmisw.org/ont/trdi/models/ZedHead
