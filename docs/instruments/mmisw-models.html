Concepts

1.1 ChannelMaster

  • Teledyne Rd Instruments model name for an ADCP designed to use horizontal profiling for flow monitoring in inland waterways.
  • Concept URI: http://mmisw.org/ont/trdi/models/ChannelMaster

1.2 HorizontalADCP

  • Instrument designed to measure velocity profiles in a horizontal plane.
  • Concept URI: http://mmisw.org/ont/trdi/models/HorizontalADCP

1.3 LongRanger

  • Model name for Teledyne RD Instruments Workhorse ADCPs of frequency 75 kHz.
  • Concept URI: http://mmisw.org/ont/trdi/models/LongRanger

1.4 Mariner

  • Model name for a Teledyne RD Instruments Workhorse Monitor ADCP configured for underway current measurement in shallow water (as opposed to the deep water Ocean Surveyor systems).
  • Concept URI: http://mmisw.org/ont/trdi/models/Mariner

1.5 Monitor

  • Teledyne RD Instruments model name for a direct reading Workhorse ADCP.
  • Concept URI: http://mmisw.org/ont/trdi/models/Monitor

1.6 Navigator

  • Teledyne RD Instruments model name for the Teledyne RDI Doppler Velocity Log.
  • Concept URI: http://mmisw.org/ont/trdi/models/Navigator

1.7 OceanObserver

  • Teledyne RD Instruments model name for a low frequency Phased Array ADCP for cabled deployment, usually from an oil rig.
  • Concept URI: http://mmisw.org/ont/trdi/models/OceanObserver

1.8 OceanSurveyor

  • Teledyne RD Instruments model name for a low frequency Phased Array ADCP for vessel mounted operations.
  • Concept URI: http://mmisw.org/ont/trdi/models/OceanSurveyor

1.9 RioGrandeADCP

  • Teledyne RD Instruments model name for an ADCP optimally configured for measurement of river discharge. A distinguishing feature of Rio Grande ADCPs is that they are designed to operate from a 12 VDC power supply (all other ADCPs operate from 20 - 60 VDC).
  • Concept URI: http://mmisw.org/ont/trdi/models/RioGrandeADCP

1.10 Sentinel

  • Teledyne RD Instruments model name for a self-contained Workhorse ADCP.
  • Concept URI: http://mmisw.org/ont/trdi/models/Sentinel

1.11 StreamPro

  • Teledyne RD Instruments model name for a miniaturized ADCP packaged into a streamlined float designed to be towed by hand across very shallow streams for quick and efficient discharge measurements.
  • Concept URI: http://mmisw.org/ont/trdi/models/StreamPro

1.12 VesselMount

  • An ADCP mounted to the hull of a vessel, typically in a sea chest, and having inputs from the vessel’s navigation equipment.
  • Concept URI: http://mmisw.org/ont/trdi/models/VesselMount

1.13 WorkhorseADCP

  • The generic model name for all of the non-phased array broadband ADCPs currently produced by Teledyne RD Instruments
  • Concept URI: http://mmisw.org/ont/trdi/models/WorkhorseADCP

1.14 ZedHead

  • Teledyne RD Instruments trade name for a transducer designed to minimize ringing.
  • Concept URI: http://mmisw.org/ont/trdi/models/ZedHead
